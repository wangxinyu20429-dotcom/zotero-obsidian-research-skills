## Multi-Idea index

When more than one Idea is created, add a lightweight index with:

- user input and common decision boundary;
- links to each Idea;
- scores and fatal-gate status;
- decisive evidence and decisive unknown for each;
- shared literature/data/project/task dependencies;
- overlap and mutual-exclusion analysis;
- recommended order and next mentor decision.

Do not duplicate the full 16-section analyses in the index.
